# 🎮 Tic Tac Toe Game

A simple Tic Tac Toe game built using HTML, CSS, and JavaScript.

## 🚀 Features
- Two-player mode (X & O)
- Automatic turn switching
- Win detection logic
- Winning animation (GIF)
- Sound effects
- Reset button

## 🛠️ Technologies Used
HTML | CSS | JavaScript
